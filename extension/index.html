Gorgias Chrome Extension¶

Gorgias is a Google Chrome extension that allows Gmail, Yahoo and Outlook users to write e-mails faster by using Autocomplete Dialog, Keyboard Completion, Alternatives and other features.

The easiest way to find out how it works is to install the extension!

Find out more at: https://gorgias.io/
